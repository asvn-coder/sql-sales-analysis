# 📊 SQL Sales & Customer Analysis

## 📌 Project Overview

This project focuses on analyzing sales, customer behavior, and product performance using SQL. The goal is to extract meaningful insights that help in business decision-making.

---

## 🛠️ Tools Used

* SQL (MySQL/PostgreSQL)
* Excel (for initial data view)

---

## 📂 Dataset

The dataset includes:

* Sales data
* Customer information
* Product details
* City-level data

---

## 🔍 Key Analysis Performed

### 1. Total Revenue Analysis

* Calculated overall sales revenue

### 2. Top Selling Products

* Identified best-performing products based on sales

### 3. Customer Analysis

* Found high-value customers
* Analyzed purchasing patterns

### 4. City-wise Performance

* Compared sales across different cities

### 5. Monthly Sales Trends

* Analyzed how sales change over time

---

## 📈 Key Insights

* Majority of revenue comes from top-performing products
* Certain cities contribute significantly higher sales
* Customer purchasing behavior varies by region

---

## 🚀 Conclusion

This project demonstrates how SQL can be used to analyze business data and generate actionable insights.
